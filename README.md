# Hi, I'm Kopi

I'm a second-year BSIT student going full-stack this year, and I build to prove it — not just study it.

I started in Python, but the feedback loop was invisible: run a script, read text in a terminal, repeat. Web development changed that. The first time I saw a change in code reflected instantly in a browser, it clicked — I could see what I was building, iterate on it, and show it to someone else immediately. I went all in from there.

Since then I've shipped Vex, a production-grade AI website critique tool with SSRF-hardened backend architecture, deterministic fact extraction to reduce AI hallucination, and a full caching and rate-limiting layer — built solo, as a second-year student. I'm not interested in "impressive for a beginner." I'm interested in code that holds up regardless of who wrote it.

Open to remote opportunities.
