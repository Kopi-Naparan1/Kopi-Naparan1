## Projects

### 🚀 [Vex — AI Website Roaster](https://website-roaster-weld.vercel.app/)

An AI-powered website critique tool that evaluates clarity, copywriting, trust, calls-to-action, and mobile experience.

**Highlights**
- Built with Next.js, TypeScript, Gemini 2.5 Flash, and Upstash Redis
- Secure website fetching with SSRF protection, DNS rebinding defense, and pinned-IP requests
- Deterministic content extraction to improve consistency and reduce LLM hallucinations
- Versioned Redis caching and rate limiting for performance and scalability
- Designed, built, and deployed independently
